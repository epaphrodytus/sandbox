Search help

  • ■ You can select the search target from "Statistics list", "List of datasets" or "Datasets".
  • ・ "Statistics list"... Statistics name, explanation of statistics.
  • ・ "List of datasets"... the ones that put datasets in a similar category together.
  • ・ "Datasets"... Meta informations (attached information) and data resources.

  • ■ Advanced Search
  • ・"Search Metadata"
  • 　... Search meta informations (attached information), such as metadata and table title.
  • ・"Search Data"
  • 　... Search data resources from database or files.

  • ■The following operators can be specified as search conditions.
  • ・" "(space) or "and" ... Pages containing all keywords are searched.
  • ・"or" ... Pages containing one of the keywords are searched.
  • ・"-" ... Pages containing the keywords are excluded from search.
  • ・Double-quotation "<keyword>"...The keyword will be searched in exact match.
  • ・Half bracket "()" ... The search condition in parentheses takes precedence.
  • 　(When specifying "and","or" there must be a space before and after.
  • 　　When specifying "-", there must be a space before.)

  • ■Enter with a combination of keywords and operators.
  • ・e.g. Population Census and Population
